Blog Ready

My blog is ready to publish! So let's start with the first article. Here are some details of the implementation.

Entries

The plan for the blog is to publish one or two articles per month, so for this first version, it is not necessary to prepare any backend service. The blog data will be stored on a JSON simulating an API, each entry of the array will have the meta tag information and a reference to a local HTML file with the article content, here is an example: 

  [

    {

      "article": "test"

      "title": "Test",

      "imgSrc": "https://www.manueme.com/images/Grid.png",

      "brief": "This is a test entry",

      "imgAlt": "",

      "date": "2019/09/01",

      "author": "Manuel Machado",

      "keywords": "Angular",

      "htmlPath": "test.html",

      "description": "Hello world"

    }

  ]

Line numbers

The central component of the application is the ide-constructor, calculates the height of its content (it could be an entry or a complete article) and adds the line numbers by dividing the total height with the line-height.

There is a line number component, an empty line component, the include header, and the constructor itself that includes all of these.

Expand and collapse animation

"Giving life" to a site is very important and I like to spend time on it so there's a couple of animations on this web app. The expand and collapse code animation when the user clicks the plus on the minus button on the left side of the "class constructor" was generated in http://animista.net/.

Spoiler alert. There's a link to my GitHub at the end of the article, but here is the ide constructor (SCSS) class: 

  .ide_constructor_body {

    height: 0;

    flex: 1 auto;

    transition: height 0.2s ease-in-out, flex-basis 0.2s ease-in-out;

    -webkit-transition: height 0.2s ease-in-out, flex-basis 0.2s ease-in-out;

    animation: swing-out-top-bck 0.2s cubic-bezier(0.200, -0.280, 0.735, 0.045) both;

    -webkit-animation: swing-out-top-bck 0.2s cubic-bezier(0.200, -0.280, 0.735, 0.045) both;

    

    &.show {

        height: 100%;

        flex-basis: 100%;

        animation: swing-in-top-fwd 0.5s cubic-bezier(0.175, 0.885, 0.320, 1.275) both;

        -webkit-animation: swing-in-top-fwd 0.5s cubic-bezier(0.175, 0.885, 0.320, 1.275) both;

    }

  }
