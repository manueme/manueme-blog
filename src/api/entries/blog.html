My blog is ready to publish! So let's start with the first article. Here are some details of the implementation.

Entries

The plan for the blog is to publish one or two articles per month, so for this first version, it is not necessary to prepare any backend service. The blog data will be stored on a JSON simulating an API, each entry of the array will have the meta tag information and a reference to a local HTML file with the article content, here is an example: 

    [

        {

          "article": "test"

          "title": "Test",

          "imgSrc": "https://www.manueme.com/images/Grid.png",

          "brief": "This is a test entry",

          "imgAlt": "",

          "date": "2019/09/01",

          "author": "Manuel Machado",

          "keywords": "Angular",

          "htmlPath": "test.html",

          "description": "Hello world"

        }

    ]
